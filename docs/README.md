## üí° Glossary

- **Agent**: An autonomous AI system that can perform tasks
- **Plugin**: A set of functions that an agent can call
- **Semantic Function**: A function defined using natural language prompts
- **Native Function**: A function written in code (C#, Python, Java)
- **Kernel**: The core orchestration engine of Semantic Kernel
- **Planner**: A component that creates execution plans
- **Memory**: Storage for context and conversation history
